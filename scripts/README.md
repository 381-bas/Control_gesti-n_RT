# Scripts

Esta carpeta contendrá los procesos reproducibles del proyecto.

## Componentes planificados

- `leer_ruta_rutero.py`: ingesta histórica y QA.
- `aplicar_vistas.py`: aplicación versionada de las vistas SQL.
- `publicar_snapshot.py`: creación de una copia SQLite estable para consumo del dashboard.

Los scripts no deben incorporar lógica visual ni modificar los Excel de origen.
